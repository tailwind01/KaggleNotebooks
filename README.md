# Notebooks from Kaggle

This repo contains the Kaggle notebooks and code-runs in Jupyter Notebooks. The commits to this repo are the ones that are given as assignments on Kaggle for machine-learning and SQL.

Like the Practice repo, these solutions/ codeblocks also need to be read in conjunction with the problem statements available on Kaggle. Further, owing to many changes since 2021, many of the assignments and the datasets may have undergone changes, but the basic principles would be the same.
